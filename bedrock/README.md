# AWS Bedrock Agent Configuration for Hustlr

# This directory contains configurations and scripts for the AWS Bedrock Agent
# that powers the conversational AI for Hustlr.

# Agent Name: HustlrAgent
# Purpose: Interpret user requests via WhatsApp and trigger appropriate actions

# Action Groups:
# - search_providers: Find service providers by type and location
# - create_booking: Create a new service booking
# - register_provider: Register a new service provider

# For development, use Quick-create Lambda functions for initial testing.
# Later integrate with the FastAPI backend for action execution.
